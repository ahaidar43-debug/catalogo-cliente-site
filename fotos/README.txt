Coloque as fotos dos produtos nesta pasta.

Forma mais simples:
- use o codigo do produto como nome do arquivo
- exemplo: PROD-00002.jpg
- depois atualize o arquivo photo-map.js com esse nome, ou me envie a pasta de fotos que eu gero esse mapa automaticamente

O app tambem aceita fotos configuradas no arquivo photo-map.js.
Exemplo:
window.PRODUCT_PHOTOS = {
  "PROD-00002": "PROD-00002.jpg",
  "PROD-00003": "minha-foto-produto.webp"
};

Formatos recomendados: .jpg, .png ou .webp.
